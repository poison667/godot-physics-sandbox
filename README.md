# Godot 4.6.3 Physics Sandbox Prototype

Full step-by-step guide with Jolt Physics integration for a Bonelab-style physics sandbox.

## Features
- Procedural textures
- Ragdoll enemies
- Grab/Punch/Shoot mechanics
- Jolt Physics enabled for better performance

See the full guide in this repository.

## Setup
1. Clone this repo.
2. Open in Godot 4.6.3.
3. Enable Jolt in Project Settings.

All scripts and instructions are provided in the conversation history or additional files.
